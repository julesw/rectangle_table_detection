Detailed Description

Main file of the node detect_normal_planes that detect (all) planes that are normal to a given vector, in a given height range.

Actually, the user can define some limits for the plane detection: -a minimal number of inliers of the plane, so too small planes are not detected. -a number of planes to detect When the first of the above limition is met, the program stops and waits for an other input point cloud.

The method used is pcl ransac method, and it detects geometrical planes; points can be not grouped, or even not define a physical plane.
