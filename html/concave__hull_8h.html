Detailed Description

2D hull computation utilities for point clouds.

Author
Jules Waldhart

Utilities using pcl::ConcaveHull, pcl::ConvexHull to computer 2D hulls of 2D point clouds. Works on XYZRGBA (kinect-like, RGB + Depth) point clouds

Compute concave hull of a plan.

Parameters
[in]cloudthe complete cloud.
[in]inliersindices of points to compute; indicates plan inliers.
[in]coefficientsthe coefficients of the plan.
[in]alphathe alpha parameter for the concave hull method (the smaller, the more detailed is the hull). Only >0 values.
Returns
the hull as a PointCloud of points representing the plan's hull.

This function firstly project inliers over the plan indicated by it's coefficients, and then compute the hull of these projected points.
