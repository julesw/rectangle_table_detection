compute ransac over the point cloud to find a plan normal to the vertical and in the height interval.

Parameters:
[in]cloudthe cloud
[in]vert_cameraa parametrized line representing the axis to use as the normal of the plan and the point at a 0 height
[in]ransac_model_epsilonthe epsilon value for the ransac plane model
[in]ransac_dist_thresholdthe maximum distance to the ransac plane model
[out]inliersthe inliers indices of the plane found
[out]coefficientsthe coefficients of the plane found
[in]min_height_planethe minimum height of the plane to search for
[in]max_height_planethe maximum height of the plane to search for

this function firstly selects all the points that are in the given range, then search for a plane definition with RANSAC method within this points selection.
